How to install dependencies and upload them to S3 bucket from terminal:
  mkdir python
  pip install --target ./python sagemaker
  sudo yum install zip
  zip -r sagemaker.zip ./python
  aws s3 cp ./sagemaker.zip s3://{YOUR_BUCKET_NAME}/
